# Visão do Produto

## Nome do Projeto
Agenda Simples

## Descrição 
O Agenda Simples é uma plataforma SaaS para agendamento de serviços, criada para facilitar a relação entre prestadores de serviço e clientes.

A proposta do sistema é reduzir o esforço manual envolvido na marcação de horários, centralizando informações sobre serviços, profissionais, disponibilidade e agendamentos em uma única plataforma.

## Problema

Muitos prestadores de serviço ainda gerenciam suas agendas manualmente, principalmente por WhatsApp, planilhas ou anotações. Esse processo pode gerar conflitos de horário, perda de informações, demora no atendimento ao cliente e dificuldade para acompanhar a rotina do negócio.

## Usuário Principal

O usuário principal do sistema é o prestador de serviço ou empresa prestadora de serviço.

Exemplos:

- Barbearias
- Clínicas
- Salões de beleza
- Consultórios
- Estúdios de tatuagem
- Pet shops
- Profissionais autônomos

## Usuário Secundário

O usuário secundário é o cliente final, que deseja visualizar serviços disponíveis e realizar um agendamento de forma simples.

## Proposta de Valor

O sistema permite que prestadores de serviço organizem sua agenda, reduzam tarefas manuais, evitem conflitos de horário e ofereçam uma experiência mais simples para seus clientes.

## Objetivo Técnico

Este projeto será desenvolvido como um projeto vitrine para aplicar conceitos de Engenharia de Software, incluindo:

- Arquitetura de software
- APIs REST
- Programação Orientada a Objetos
- Princípios SOLID
- Banco de dados relacional
- Testes automatizados
- Documentação técnica
- Observabilidade
- Boas práticas de versionamento
- Deploy em ambiente cloud
